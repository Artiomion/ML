# Детектор улыбки, открытых глаз и счётчик упражнений (Hands Up + Claps)

Проект реализует систему компьютерного зрения в реальном времени, которая одновременно:
- Определяет улыбку и интенсивность улыбки (в процентах)
- Проверяет, открыты ли глаза
- Считает количество поднятий рук вверх
- Считает количество хлопков
- Визуализирует ключевые точки лица и тела

### 1. MediaPipe Face Mesh – точная детекция улыбки и глаз
Самый точный и быстрый современный подход.

**Особенности:**
- 468 точек лица
- Нормированная метрика улыбки (0–100%)
- Определение открытости глаз по расстоянию век
- Отображение сетки лица и контуров губ/глаз

![MediaPipe Face Mesh](img/img_1.png)

---

### 2. dlib 68-точечная модель лица
Классический подход с предобученной моделью `shape_predictor_68_face_landmarks.dat`.

**Особенности:**
- EAR (Eye Aspect Ratio) для определения закрытых глаз
- Комбинированная метрика улыбки
- Отображение всех 68 точек и номеров
- Convex hull маска лица

> Требуется скачать файл `shape_predictor_68_face_landmarks.dat`

![dlib 68 points](img/img_2.png)

---

### 3. MediaPipe Pose – счётчик поднятий рук и хлопков
Отслеживание позы тела для фитнес-приложений.

**Функционал:**
- Счётчик поднятия обеих рук выше плеч
- Счётчик хлопков (когда запястья близко)
- Простая конечная машина состояний

---

### 4. Комбинированное решение (Face Mesh + Pose)
Объединяет всё лучшее:
- Улыбка + открытость глаз (Face Mesh)
- Подсчёт упражнений (Pose)
- Одновременная визуализация лица и тела

Идеально подходит для интерактивных игр, обучения, реабилитации и т.д.

### 5. Haar Cascade (OpenCV) – классический метод
Самый лёгкий по зависимостям (только OpenCV).

**Особенности:**
- Каскады Хаара для лица, глаз и улыбки
- Не требует дополнительных моделей (кроме xml-файлов из OpenCV)
- Работает, но менее точно и чувствителен к освещению/углу
